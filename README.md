# Snake_water_gun
A simple snake water gun game built using python,
where user plays aginst the computer
## Features of the game
-5 rounds of game
-Random computer choices
-Invalid input handling
## Rules to play
-snake drinks water : snake wins
-gun kills the snake: gun wins
-water drowns the gun: water wins
## Technologies used
-python
-random module
-dictionaries
-conditional statements
## How to read
python main.py
